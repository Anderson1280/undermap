# Política de Segurança

## Versões suportadas

| Versão | Suporte |
|--------|---------|
| 0.1.x  | ✅ Ativa |

## Reportar uma vulnerabilidade

**Não abra uma Issue pública** para reportar vulnerabilidades de segurança.

Envie um e-mail para **security@undermap.com.br** com:

- Descrição detalhada da vulnerabilidade
- Passos para reproduzir
- Impacto potencial estimado

Respondemos em até **48 horas** e coordenamos o disclosure responsável.

## Boas práticas para usuários

- **Nunca compartilhe seu arquivo `.env`** — ele contém suas chaves de API
- **Nunca commite credenciais** — use sempre variáveis de ambiente
- **Revogue chaves comprometidas imediatamente** em console.cloud.google.com
- Mantenha o Undermap **sempre atualizado** (`pip install --upgrade undermap`)
